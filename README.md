
## Description

Sample with NestJs controller and Event Emitter
demonstrated Usage of Cursor AI to generate Test cases, fix issues etc to imporve code quality and minimize development time


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
